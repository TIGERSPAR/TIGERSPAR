# SUG Phone Accessories - Website

A professional, modern e-commerce catalog website for selling phone accessories in Buea, Cameroon.

## Features

✅ **Product Catalog** - Display all phone accessories with descriptions and prices
✅ **Professional Design** - Modern dark theme with cyan accents
✅ **Mobile Responsive** - Works perfectly on all devices (mobile, tablet, desktop)
✅ **WhatsApp Integration** - Direct order messaging via WhatsApp
✅ **Quick Order Buttons** - One-click WhatsApp ordering (no forms needed)
✅ **NO JavaScript** - Pure HTML & CSS only (super fast & simple)
✅ **Fast & Lightweight** - No dependencies, zero scripts
✅ **Easy to Deploy** - Works on Netlify, GitHub Pages, or any static hosting

## How to Deploy

### Option 1: Netlify (Recommended - Free & Easy)

1. Go to https://netlify.com
2. Sign up with GitHub or email
3. Drag & drop the project folder into Netlify
4. Your site goes live instantly!
5. You get a free subdomain (e.g., sug-accessories.netlify.app)
6. Optional: Add a custom domain

### Option 2: GitHub Pages (Free)

1. Create a GitHub repository
2. Push all files to the repository
3. Go to Settings → Pages
4. Select "Deploy from branch" → main
5. Your site is live at `username.github.io/repo-name`

### Option 3: Any Web Host

1. Upload all files to your web host via FTP
2. Make sure `index.html` is in the root directory
3. That's it!

## Customization

### Update Prices
Edit `pages/products.html` and change the prices in each product card.

### Update WhatsApp Number
Replace `+237670959751` with your WhatsApp number in:
- `index.html` (WhatsApp button & links)
- `pages/about.html` (WhatsApp button)
- `pages/contact.html` (contact info & quick order links)
- `pages/products.html` (all product order buttons)

### Change Colors
Edit `css/style.css` - modify the CSS variables at the top:
```css
:root {
    --primary-color: #1a1a1a;      /* Dark background */
    --accent-color: #00d4ff;        /* Cyan/turquoise */
    --success-color: #25d366;       /* WhatsApp green */
}
```

### Add/Remove Products
1. Open `pages/products.html`
2. Copy a product card section
3. Paste and update with your product details
4. Update price and emoji icon
5. Update the WhatsApp link with the product name
